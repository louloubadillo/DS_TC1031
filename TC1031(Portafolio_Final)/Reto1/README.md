## Estructura de los datos
Los datos son leídos de un archivo `csv` con los siguientes campos

0. Fecha (DD/MM/AA)
1. Hora (HH:MM:SS, 24)
2. IP Fuente
3. Puerto Fuente
4. Hostname Fuente
5. IP Destino
6. Puerto Destino
7. Hostname Destino
